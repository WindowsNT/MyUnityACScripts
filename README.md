# My Unity/Adventure Creator Scripts
A set of scripts for Unity/Adventure Creator. Created by me and Claude :)


# Mesh

A script that will create a navigation based on an image mask, so the player can walk on it.

[<img src="1.jpg" width="350"/>](1.jpg)
[<img src="1.png" width="350"/>](1.png)

[<img src="result.png" width="350"/>](result.png)

Usage: Create the collision cube in AC, then add the "mesh" script as component to it.
Supports baking to file, Input from mask or polygon, Auto position the player.

# HotspotCameraGizmoFilter

Put this to the "Hotspot" root as component. Allows hotspots to be visible as gizmos only when their camera is selected.

# BackgroundImageChanger

Put this to BackgroundImage components to allow automatic image switchign (say day/night) based on a global variable.
